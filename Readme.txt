Для запуска:
Скачатьданные из репозитория
В корне проекта прописать в консоль: npm install
В app.js прописать ссылку на базу данных mongodb (у меня та же самая, что на паре)
Написать в консоль: npm run devstart
В браузере перейти на localhost:3000
